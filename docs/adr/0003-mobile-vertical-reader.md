# ADR 0003: スマホ専用の縦スクロールリーダーにする

- 状態: 採用
- 日付: 2026-07-18

## 背景

閲覧者はスマートフォンで漫画を連続して読む。初期版で PC 向けの見開き、横送り、
画面サイズごとの複数のリーダー操作を同時に設計すると、確認範囲が広がる。

## 決定

- 初期版はスマホ専用の UI とする。
- 漫画ページは一枚ずつ画面幅に収め、上から下へ連続して表示する。
- リーダーは URL の期限切れ時に巻 API を再取得し、画像を復帰させる。
- PC 向けの最適化、見開き、横送りは初期スコープに含めない。

## 理由

一般的なスマートフォンの片手操作と相性がよく、ページ順の確認、画像の再読込、
ネットワーク失敗時の復帰を単純に検証できる。
