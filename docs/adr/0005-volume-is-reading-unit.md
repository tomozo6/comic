# ADR 0005: 巻をページ列の閲覧単位にする

- 状態: 採用
- 日付: 2026-07-18

## 背景

閲覧者は巻を選んだら、途中で章・話を意識せず、その巻に収録された内容を最後まで
連続して読みたい。

## 決定

- 本システムの閲覧カタログは `作品 → 巻 → ページ` の3階層にする。
- 章・話はカタログ、API、画面のいずれにも持たない。
- 巻は順序付きのページ列を直接持つ。
- 巻の取得 API は、その巻のすべてのページと各ページの一時画像 URL をページ順で返す。

## 影響

- 巻のリーダーは、収録されたすべてのページを縦に連続表示する。
- 将来 SQLite を導入するときも、作品・巻・ページの関係だけを保存する。
- 既存の章 API は巻 API に置き換える。
