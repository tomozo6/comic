# ADR 0004: 作品詳細は巻一覧として表示する

- 状態: 採用
- 日付: 2026-07-18

## 背景

作品を選んだ後の画面は、閲覧者が読みたい単位を直感的に選べる必要がある。
作品内のページを、閲覧者が選びやすいまとまりとして整理する必要がある。

## 決定

- 作品詳細画面は巻一覧として表示する。
- 各巻のカードには表紙、巻番号、巻タイトルを表示する。
- 収録話数は画面に表示しない。
- 巻を選択すると、その巻の先頭ページからリーダーを開く。

## 影響

- カタログは作品とページの間に巻を持つ。
- 作品詳細 API は巻の一覧を返す。
- 巻が順序付きのページを直接管理する。
