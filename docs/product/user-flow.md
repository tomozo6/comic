# ユーザーフロー

v1 の閲覧導線は、次の4画面だけに絞ります。

```text
ログイン → 作品一覧 → 巻一覧 → 漫画リーダー
```

## 画面ごとの役割

| 画面 | 利用者がすること | 次の画面 |
| --- | --- | --- |
| ログイン | Google アカウントでログインする | 作品一覧 |
| 作品一覧 | 読みたい作品を選ぶ | 巻一覧 |
| 巻一覧 | 読みたい巻を選ぶ | 漫画リーダー |
| 漫画リーダー | 縦スクロールで読む | 巻一覧へ戻る |

章・話はカタログ上で管理しても、v1 の画面には表示しません。巻を選ぶと、その巻の先頭ページから読み始めます。
