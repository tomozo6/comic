# 用語集

## 漫画画像バケット

実際の漫画画像を保存する、GCP プロジェクト `tomozo6` の非公開 GCS バケット
`tomozo-manga-images`。ブラウザから直接公開せず、将来的にはアプリケーションが
発行する短命の GCS 署名付き URL でのみ読み取る。

## 一時 URL

漫画画像をブラウザへ表示するために、巻 API が返す有効期限付きの URL。ローカルでは
アプリケーションが検証する URL、本番では GCS 署名付き URL を指す。

## ローカル配信 URL

ローカル開発時に Go アプリケーションが発行する一時 URL。期限と対象画像を
アプリケーション側で検証してから画像を配信する。GCS の V4 署名 URL ではない。

## ダミー画像

実際の漫画の代わりに、開発・テスト用として使用する公開可能な画像。個人情報や
著作権上の制約がある実作品は含めない。

## 巻

作品内で閲覧者が選ぶ単位。巻番号、巻タイトル、表紙を持つ。
順序付きのページを直接持つ。初期版の作品詳細画面は巻を一覧表示する。
